# SmartRetail Analytics

Sistema de análisis de ventas y comportamiento de clientes desarrollado con Python, SQL y visualización de datos utilizando un dataset de ventas de hipermercado.

## Descripción

Este proyecto implementa un análisis comercial completo orientado a Business Intelligence, permitiendo identificar patrones de ventas, regiones con mayor rendimiento, productos más rentables y comportamiento de clientes mediante dashboards y consultas SQL.

## Tecnologías Utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- SQL
- PandasQL
- Google Colab
- KaggleHub

## Funcionalidades

- Análisis exploratorio de datos
- Limpieza y transformación de datos
- KPIs comerciales
- Visualización interactiva
- Consultas SQL
- Dashboard de ventas
- Análisis temporal
- Segmentación comercial

## KPIs Generados

- Ventas Totales
- Ganancia Total
- Promedio de Ventas
- Regiones con más ventas
- Categorías más rentables
- Productos más vendidos

## Resultados Principales

- Ventas totales superiores a 2.2 millones
- Identificación de regiones con mayor rendimiento comercial
- Detección de subcategorías más rentables
- Análisis de tendencias temporales de ventas
- Identificación de clientes con mayores compras

## Dataset

Dataset utilizado:
Superstore Dataset (Kaggle)
